# 🔐 ANUKHRA: The Cryptic Hunt

An interactive web-based puzzle experience where users decode hidden clues across multiple formats — including text, images, audio, and QR codes.

> This isn’t just a quiz. It’s a layered challenge where each answer unlocks the next.

## 🏗️ How It Works

1. User starts at the entry level
2. Each puzzle contains a hidden clue
3. Correct answer unlocks the next stage
4. Difficulty increases progressively
5. Multiple thinking styles are required (logic, observation, decoding)
